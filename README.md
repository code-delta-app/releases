# CodeDelta — release bundles

Public download home for CodeDelta. Each release carries the **Linux engine bundle**
(`codedelta-linux-full.tar.gz`) — the engine that the
[GitHub Action](https://github.com/code-delta-app/action) downloads and runs.

The Action pulls it from the **latest** release automatically:

```
https://github.com/code-delta-app/releases/releases/latest/download/codedelta-linux-full.tar.gz
```

Desktop apps (macOS / Windows) are distributed separately via **https://codedelta.app**.
